- venv
  Entorno aislado donde puedes instalar librerías sin afectar la instalación global de Python

- Crear y activar un entorno virtual
  python -m venv venv
  venv\Scripts\activate

- Archivo de dependencias (instalar dependencias)
  pip install -r requirements.txt

- Ejecutar
  - py software.py ó phyton software.py
